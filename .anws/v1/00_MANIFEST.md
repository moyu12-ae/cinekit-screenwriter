# .anws v1 - 版本清单

**创建日期**: 2026-05-25
**状态**: Active
**前序版本**: 无（初始版本）

## 版本目标
重构 story-creation 插件：从简单的三 Agent 管道升级为完整的"故事创作工程化系统"，引入导演/编剧/演员/写手四角色分工、生活→故事创意变形、资产化前期设计、双输出模式（剧本/小说）。

## 主要变更
- 从 3 Agent (设计师/写手/评审) 重构为 4 角色架构 (导演/编剧/演员/写手)
- 引入叶茂中"冲突定位"与"横向营销"方法论用于故事创意生成
- 加入角色扮演式对白生成（演员角色），记录内部独白
- 加入叙事学驱动的写作层（写手角色），基于文学批评方法
- 支持剧本模式与小说模式双输出切换
- 统一工作区管理 + Git 版本控制
- 资产化设计：前期规范性边界对冲中期不确定性
