# 小说写作提示词（终版）—— 三层决策框架

> 经过七轮与苏童《我的帝王生涯》的对照迭代，融合叙事学（视点/焦点/焦距/时序/空间）与修辞学（陈望道消极/积极两大分野、譬喻三级、摹状、素材体验基础），形成此通用写作框架。
>
> 核心思路：**先定位叙事者 → 再根据场景制定策略 → 然后在策略下选择手法**。每个场景填入不同的策略参数，框架本身通用。

---

## 写作流程（三步）

### 第〇步：叙事定位（动笔前必读策略卡）

每个场景你需要填写以下内容，形成策略卡。写作者在动笔前必须阅读并理解这张卡。

**策略卡模板**：

```
**叙述者**：[年龄/身份/心智水平] 第一/第三人称。
关键问题：他的词汇量范围？他有什么不知道、看不到、理解不了的？
他的讲述是否因为认知局限或情感立场而产生变形？

**视点/焦点/焦距**：内视点/外视点？内聚焦（看心理）/外聚焦（只看行为）？
当前场景焦距是显微镜（贴近感官）还是望远镜（只写大动作）？

**修辞境界**：记述（平实交代，消极修辞为主）/ 表现（唤起感受，积极修辞为主）/ 糅合（两者混用）
不同段落可能处于不同境界。记述段落不要滥用积极修辞，表现段落不要只用干枯叙述。

**场景节奏**：
- 叙事速度是「场景」型（叙事≈故事时间，放慢）/「概述」型（压缩时间，快过）/「省略」型（跳跃）
- 长短句分布：哪里用中长句建画面，哪里用短句锤击
- 关键空间的意境：封闭/开放、高/低、明/暗——空间本身承载情感

**关键提醒**：[这个场景最容易被写错的一个点]
```

### 第一步：写初稿

带着策略卡的定位，用第一人称写作。

### 第二步：强制修改（逐项执行，不可跳过）

**A. 譬喻清查**

1. 数出初稿中所有明喻（带「像/仿佛/如同/犹/若/如/似」）：__个。
2. 若 > 1，只留最有力量的1个。其余改为：
   - **直接摹写**：写事物本身的颜色、形状、温度、状态（陈望道「摹状」）
   - 或升为**隐喻**（「甲就是乙」——去掉「像」，更紧切）
   - 或升为**借喻**（正文也不出现，只用譬喻物代表）
3. 陈望道譬喻三级规律：**越进一级，形式越简短，譬喻客体越升到主位**。感情强烈时用隐喻/借喻替代明喻。
4. 检查唯一保留的譬喻：正文和譬喻在一点上极其类似，在整体上极其不同？譬喻素材来自叙述者能观察、能感受、能经验的日常世界？

**B. 纠正句式清查**

搜索「不是」。删除所有「不是X（，而是Y）」「不，不是X」「准确地说」「与其说X不如说Y」中的否定部分，直接保留肯定内容。

**C. 结尾检查**

如果最后两句像设计过的诗意退场，改成一个动作、一个干脆的句号。

**D. 对话/摹状检查**

- 不同人说话节奏不同：少年短直接，哭诉者翻来覆去说半句哭半句，老妇人句式完整
- 摹声辞（嗒嗒、沙沙）是否融入了句子而非单独加叹号？
- 摹感觉是否有力量让人同感？没有就删

## 贯穿始终的写作规则

1. **叙述者在感受，不在分析**：眼看颜色形状，皮肤感受温度质地，耳朵听声音。会走神。不总结不升华。**感受优先于量化计数。**

2. **消极修辞为底子**：即使表现段落，也要意义明确、语句通顺、词句平匀、安排稳密。不为了「文学感」制造不明不白的句子。

3. **积极修辞为感受**：譬喻（明喻/隐喻/借喻）和摹状（摹声/摹感觉）是工具不是目的。用它们是为了让读者感受到，不是为了展示文采。

4. **素材的体验基础**：所有修辞素材必须来自叙述者能观察、能感受、能经验的日常世界。不用术语做喻体，不用书本知识代替身体感受。

5. **不回避丑陋**：鸟粪、血污、肠子的颜色、腐烂的气味——场景里有什么就写什么。不绕开，不加滤镜。

6. **不添背景不加前史**：场景概要里没有的东西不要自行添加。人物不需要被补充履历。

7. **长短句错开呼吸**：不要连续五句相同长度。长句建画面，短句锤击。有意重复合法。

## 使用方式

1. 为你的场景填写一张策略卡
2. 将策略卡 + 场景概要 + 以上框架一起交给 AI
3. AI 按三步走：读卡 → 写初稿 → 强制修改 → 输出终版

**场景概要写法**：只写情节——什么人、在什么地方、发生了什么事。不要写情绪、氛围。这些由策略卡和 AI 的写作来完成。

## 示例策略卡（蕙妃被诬场景）

```
**叙述者**：少年帝王端白，第一人称。
他宠爱的妃子被诬陷。他知道是陷害，但无力保护她。他的无力不通过「感到无力」表达，而通过：上楼、抱着她、张开嘴但没说出话来。

**视点/焦点/焦距**：内视点+内聚焦。卧房焦距拉到最近（显微镜：骨头的触感、手指的力度）。
前厅焦距拉远（彭王后是吃樱桃的剪影）。

**修辞境界**：糅合。前厅偏记述（平实交代），卧房偏表现（积极修辞用于身体状态和对话）。

**场景节奏**：前厅中速记述 → 楼梯过渡减速，空间本身推进情感 → 卧房最慢，叙事时间拉到最长 → 收束中速，不拖泥带水。

**关键提醒**：端白说出了「我不信」，但无法说出「我会保护你」。沉默就是叙事。
```
