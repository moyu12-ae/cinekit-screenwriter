# 第二轮对比分析与经验记录

## V1 → V2 改进幅度总结

| 维度 | V1 状态 | V2 状态 | 改进 |
|------|---------|---------|------|
| 明喻密度 | 每段5-6个 | 每段1-2个 | ★★★★ |
| "不是/而是"变体 | 明确存在 | 基本消除，偶有边界情况 | ★★★ |
| 叙述者成人化 | 回顾性理解 | 基本消除 | ★★★★ |
| 替读者解释 | "零下二十度" | 消除 | ★★★★★ |
| 诗意结尾 | 文艺腔收束 | 干脆、简单的收束 | ★★★★ |

| 走神即是性格 | 无 | 充分体现 | ★★★★★ |
| 对话自然度 | 功能化 | 明显更自然 | ★★★ |

## V2 相比 V1 的具体进步

### 进步 1：少年叙述声音的出现

V1 版本的叙述者是一个成年文学作家在模仿少年。

V2 版本中出现了真正属于14岁孩子的思维：
- "我数了，七声咔。然后轿子拐进了德奉殿前的广场。台阶一共三十九级。我一级一级数完了。"
- "父亲在里面炼过七十七粒丹。现在炉子冷了。没人添柴了。说不定以后也不添了。"
- "鹂鸣两个字是我取的。黄鹂的鹂。"

这些句子的力量在于——它们不解释、不评价、不文学化。它们就是一个孩子在观察、在计数、在回忆。

### 进步 2：数字作为节奏工具

V1 完全没有使用数字。

V2 中数字成为核心节奏手段：
- 数台阶（39级）
- 数丹药（77粒）
- 数樱桃核（3个）
- 数尸体（第17具...第26具...第37具...第38具）

数字让叙述有了客观的框架，读者在这个框架中自己感受。这和苏童「我清点了一下莜麦地里的死尸，一共是三十七具」的手法高度一致。

### 进步 3：结尾的力量

V1 结尾：「露水从琉璃瓦上滑下来，打在我的靴面上，隔了很久，又一滴」——设计出来的诗意。

V2 结尾：
- 段落1：「我伸手接了一片。黑的。不热。」
- 段落2：「我把弓放下，弓身上沾了一层薄薄的血迹，不知道是什么时候蹭上的。我拿袖子擦了擦，擦不掉。」
- 段落3：「天正在亮起来。」

V2 的结尾不再追求「漂亮」。段落1 的结尾「黑的。不热」——两个短句、三个字词，像两锤敲在读者心上。段落3 的结尾「天正在亮起来」——简单得几乎什么都没说，但这就是对的。蕙妃走了，天亮了，生活继续。

### 进步 4：彭王后吃樱桃

这是 V2 段落3 中最高光的时刻。提示词 V2 中把苏童的樱桃细节作为示例写了进去，而子 Agent 不仅学会使用这个细节，还延伸出了自己的创造：

「碟子里有三个核。」

这三颗核——被数出来——比「她一直在吃」有力一百倍。这就是提示词中「数字是呈现工具」这条规则被真正理解的证明。

## V2 仍然存在的差距

### 差距 1：句式碎片化的边界

V2 大量使用短句和碎片句（「不热」「黑的」「没有眼珠」「话没出来」），这是苏童也用的手法。但 V2 的碎片化程度在某些地方超过苏童——苏童的碎片句之间有更长的句群做缓冲和铺垫。

### 差距 2：对话中的人物辨识度

苏童原文中，蕙妃的哭诉有非常独特的节奏和重复（「一只白狐，一只白狐」「怎么小心都没有用」）。V2 的对话虽然自然了，但人物的语言辨识度还不够高——每个角色的说话方式还不够「只属于他/她」。

### 差距 3：「不避讳」还不够彻底

V2 仍然没有出现苏童式的完全不对丑物加滤镜的描写。苏童写「那只手形如黑饼，上面溃烂的血痂招来了一群苍蝇」「嘴角流出一条口水的黏液」——这种绝对的生理性的不回避，V2 还没有完全做到。

### 差距 4：意象的贯穿性

苏童原文中「纸人」意象贯穿多个场景（蕙妃像纸人→后妃们也是纸人→终有一天被大风卷走），形成了一种结构性意象。V2 每个场景各自使用意象，但没有跨场景的贯穿。

## 经验汇总（两轮迭代的核心教训）

### 经验 1：规则从「禁止」到「示范」的转变
V1 的规则主要是「不要做什么」。V2 加入了苏童的具体手法作为正向示范（樱桃、数字、走神）。实验证明，正向示范比禁止性规则有效得多。

### 经验 2：叙述者年龄是文笔的锚点
一旦把叙述者的年龄（14岁）和他的认知能力绑定，很多「AI腔」自然就消失了。因为14岁的孩子不会替读者解释、不会做诗意的总结、不会用「后来我才明白」这种回顾性句式。

### 经验 3：明喻是AI的默认模式
AI在描写事物时的第一反应永远是「找一个别的领域的东西来比喻」。要打破这个默认模式，需要同时做两件事：①硬性限制数量（每场景最多1个）②要求喻体来自场景内部语义域。

### 经验 4：结尾是AI最容易暴露的地方
AI有强烈的「完形冲动」——要把一个段落包装成一个完整的、有头有尾的、有韵味的单元。小说不是这样的。小说的段落是流的片段，不是独立的诗。

### 经验 5：最好的规则是给例子
提示词中「一个具体动作胜过十句心理描写」加上了苏童樱桃的例子（V2），子Agent立刻学会了这个模式。而同样的规则在没有例子的V1中被忽略了。
