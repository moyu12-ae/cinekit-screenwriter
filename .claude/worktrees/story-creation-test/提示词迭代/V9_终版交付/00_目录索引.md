# V9 小说写作提示词 —— 终版交付

## 文件夹内容

| 文件 | 内容 | 用途 |
|------|------|------|
| `01_三Agent流水线框架.md` | 完整的导演→写手→编辑三Agent提示词模板，可直接复制使用 | **日常写作时使用** |
| `02_九轮迭代全记录.md` | V1到V9的完整迭代历史，每轮做了什么、发现了什么、为何改进 | 理解框架的设计逻辑 |
| `03_策略卡模板库.md` | 叙事定位、时距分配、修辞境界、空间策略、质感分析、分镜策略的填写模板和示例 | 为新场景快速生成策略卡 |
| `04_Skill扩展路线图.md` | 从当前提示词到完整Skill的四阶段规划，包含架构设计、自动化方案 | 后续开发参考 |
| `05_完整测试案例.md` | 三个测试场景从输入到终版的完整流水线记录 | 验证框架有效性；作为新场景的参考 |

**相关 Skill**：
- `style-extractor`（文章风格提取器）：独立 skill，从任意中文文学文本中提取风格特征，产出作家风格画像。画像可直接作为 V9 流水线中导演 Agent 的质感分析输入。位于 `写作 skills/style-extractor/`。

## 快速使用

1. 为你的场景编写**动作/对白骨架**和**分镜策略初稿**（参考 `03_策略卡模板库.md`）
2. 将骨架+分镜策略交给 **Agent A（导演）**，产出完整执行方案（参考 `01_三Agent流水线框架.md` 中的 Agent A 提示词）
3. 将导演方案交给 **Agent B（写手）**，产出描写层正文
4. 将写手正文+质感参考交给 **Agent C（编辑）**，审核质量
5. 如编辑打回，将修改意见交给写手修改，重新提交编辑

## 核心架构

```
输入：动作/对白骨架 + 分镜策略 + 质感参考文段
        │
   ┌────▼──────────────────────────────────────┐
   │  Agent A：导演（DeepSeek V4 纯分析模式）     │
   │  ①叙事定位 ②时距策略 ③境界分配             │
   │  ④空间策略 ⑤质感分析 ⑥分镜脚本             │
   └────┬──────────────────────────────────────┘
        │
   ┌────▼──────────────────────────────────────┐
   │  Agent B：写手（角色沉浸模式）              │
   │  第一人称，按分镜转化描写层                  │
   └────┬──────────────────────────────────────┘
        │
   ┌────▼──────────────────────────────────────┐
   │  Agent C：编辑（纯分析模式）                │
   │  抄袭检查 + 明喻/纠正/结尾/景别审核          │
   │  通过→输出 / 打回→写手修改                  │
   └────┬──────────────────────────────────────┘
        │
     最终正文
```

## 知识来源

- 陈望道《修辞学发凡》：消极/积极修辞两大分野、譬喻三级（明喻→隐喻→借喻）、摹状（摹声/摹感觉）、素材体验基础
- 汤拥华《文学批评入门》+ 麦基《故事》：叙述者定位、视点/焦点/焦距、叙述时间五时距、叙述空间三维度
- 罗恩·罗泽尔《这样写出好故事：描写与背景》：呈现/叙述、多余法则、最好的描写是不描写、感官描写、句子变化
- DeepSeek V4 角色扮演指令：纯分析模式 vs 角色沉浸模式
- 苏童《我的帝王生涯》：目标文本质感参照
