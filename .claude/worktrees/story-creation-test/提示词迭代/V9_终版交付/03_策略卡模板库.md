# 策略卡模板库

> 为 Agent A（导演）准备的可复用模板。每个场景填入不同参数。

---

## 模板1：叙事定位卡

```
### 第一步：叙事定位

**叙述者**：[年龄/身份/心智水平] 第一人称「我」。
- 他知道什么：[列出]
- 他不知道什么：[列出，因认知有限/信息不对称/情感防御]
- 他的感知有什么变形？[如：放大感官细节，压缩人际信号；在极大压力下注意力收缩到微小物体]
- 不可靠度评估：[可靠/部分不可靠] —— [原因：认知局限/情感立场/有意隐瞒？]
  读者应该在哪里看到真相：[叙述裂缝的具体位置]
```

### 填写示例（蕙妃被诬场景）

```
**叙述者**：少年帝王端白，第一人称「我」。
- 他知道什么：白狐是栽赃。蕙妃被后宫势力联合迫害。自己身为皇帝却无力保护她。
- 他不知道什么：蕙妃内心究竟有多绝望。后妃们的具体计划。蕙妃出宫后是否安全。
- 他的感知有什么变形：他对蕙妃的描写倾向于诗意化——越接近死亡越圣洁。对彭王后的憎恶被压制为冷静剪影。
- 不可靠度评估：部分不可靠（因情感卷入）。
  读者应该在哪里看到真相：他叙述自己的沉默和拥抱，但从不承认懦弱。叙述裂缝在于——他能做的是"抱着她"而不是"保护她"。
```

## 模板2：叙述时间策略卡

```
### 第二步：叙述时间策略（时距分配）

| 段落/阶段 | 时距类型 | 内容概要 | 选择理由 |
|-----------|---------|---------|---------|
| 1. [段落名] | [省略/概述/场景/拉伸/停顿] | [一句话] | [为什么用这个时距？] |
| 2. ... | | | |

时距节奏曲线：[如：停顿→场景→概述→场景→拉伸→场景→停顿]
整体节奏描述：[如："从静到动、从慢到快再到卡住的曲线"]
```

### 时距类型选择指南

| 时距类型 | 何时使用 | 示例 |
|---------|---------|------|
| **省略** | 跳过不重要的时间段；制造冷酷感（"他没有再看"） | "龙辇继续往西走"——省略了杨松死后龙辇驶过的距离 |
| **概述** | 压缩重复性事件；建立钝感；过渡 | "从第一具数到第三十七具"——压缩数尸体的过程，建立端白的疲惫 |
| **场景** | 核心对话和动作；需要读者同步体验的瞬间 | 蕙妃与端白的卧房对话——每一秒都不能跳过 |
| **拉伸** | 放大关键时刻；主观时间膨胀；让读者凝视 | 端白走神看玉如意上的灰线——在极大压力下时间变慢 |
| **停顿** | 建立氛围；环境描写；悬停制造静谧 | 近山堂晨景——在父亲死讯到达前，时间暂停 |

## 模板3：修辞境界分配卡

```
### 第三步：修辞境界分配

| 段落/阶段 | 境界 | 理由 |
|-----------|------|------|
| 1. [段落名] | [记述/表现/糅合] | [为什么用这个境界？] |
| 2. ... | | |

境界谱系：[如：糅合→记述→糅合→记述→表现→表现→糅合]
整体倾向：[如：记述占多数——叙述者的策略是"我只说我看到的"]
```

### 境界选择指南

| 境界 | 特征 | 何时使用 | 典型错误 |
|------|------|---------|---------|
| **记述** | 消极修辞为主，平实交代，以"明白"为目标 | 过渡、背景交代、快速叙事 | 用比喻去交代"他很累" |
| **表现** | 积极修辞为主，唤起感受，以"感染"为目标 | 核心场景、情感高潮 | 修辞堆砌但无感染力 |
| **糅合** | 记述骨架+表现意象 | 大部分小说段落 | 境界不分，记述段落滥用辞格 |

## 模板4：空间策略卡

```
### 第四步：空间策略

**叙述者的空间位置**：[高处/低处/中间？固定/移动？]
**空间移动轨迹**：[由外而内？由下而上？由远而近？画出空间流线]
**空间转换标志**：[每个空间转换的标志是什么？光线的变化？声音的变化？]
**空间意境**：
- 空间A：[封闭/开放] [明/暗] [冷/暖] —— 承载什么情感？
- 空间B：...
**人物在空间中的位置**：[比例关系——人物在多大的空间里？]
```

### 填写示例（父王驾崩场景）

```
**叙述者的空间位置**：开篇在近山堂前石案旁（水平空间），被抬入轿子（封闭暗箱），抵达德奉殿台阶下，上台阶（垂直空间），止于殿内。
**空间移动轨迹**：水平移动（轿子）→垂直上升（台阶）→静止（殿内）→空间被压缩（冠压住视角）。
**空间转换标志**：近山堂→轿子：鹭鸟消失；轿子→德奉殿：白色淹没视野；台阶→殿内：光线收窄。
**空间意境**：
- 近山堂：冷、静、空——属于14岁孩子的自然世界
- 轿子：狭、暗、晃——过渡态
- 德奉殿：沉、热、挤——权力仪式空间的压迫感
- 端白自身（最终空间）：重、卡、低不下去——权力以重量的形式进入身体
```

## 模板6：分镜策略卡

```
### 第六步：分镜策略

| 镜号 | 景别 | 内容概要 | 目的 | 感官通道 |
|------|------|---------|------|---------|
| 1 | [全景/中景/特写] | [一句话] | [建立氛围/推进叙事/冲击/收束] | [选1-2个] |
| 2 | ... | | | |
| ... | | | | |

景别分布统计：全景__个 中景__个 特写__个（特写占比__%）

景别设计自检：
- [ ] 没有连续两个特写——中间有中景或全景呼吸
- [ ] 没有连续两个全景——中间有特写锚点
- [ ] 特写占比不超过40%
- [ ] 上一个镜头的落点连接下一个镜头的起点
```

### 景别选择指南

| 景别 | 画面比例 | 叙事功能 | 句式建议 | 感官建议 |
|------|---------|---------|---------|---------|
| **全景/远景** | 人物在环境中是小的 | 建立氛围、空间关系、过渡 | 中长句为主 | 视觉为主（光线、色彩、空间层次） |
| **中景** | 人物占据画面一半 | 叙事推进、人物互动、对话 | 中句为主 | 视觉+听觉 |
| **特写** | 某个细节填满画面 | 情感冲击、关键物象、身体感受 | 短句或中句 | 触觉+视觉 |

### 镜头衔接的常见模式

- **全景→中景**：从环境进入人物行动（"大殿上站满了人。祖母拉住我的手。"）
- **中景→特写**：从人物互动切入关键细节（"她抓住我的腰带。指节骤白，布帛深勒进虎口。"）
- **特写→全景**：从一个物象拉开到全景空间（"地上有一只鞋。马车在甬道上越来越小。"）
- **特写→特写**（禁止连续两个）：需中间插入中景或全景呼吸
