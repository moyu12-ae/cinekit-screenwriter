# 第一轮对比分析与经验记录

## 段落 1：父王驾崩

### AI版本关键问题

| 问题类型 | AI文本 | 问题说明 |
|---------|--------|---------|
| 说明文残余 | "不，不是鹤" | "不是...而是"的微型变体，叙述者在自我纠正 |
| 说明文残余 | "不，更像一截断了的指节" | 同上，又是自我纠正 |
| 替读者解释 | "后来我才明白那是指甲印" | 直接在解释，把读者的工作做了 |
| 文艺腔句子 | "霜正化去，阶上的湿印子像谁家的孩子用指头蘸水画下的" | 过度诗意化，叙事者不应该突然变成一个文艺诗人 |
| 添加无据细节 | "冠是新的，皮革的气味还没散尽" | 这个细节合理但属于自行添加，不在概要中 |

### 苏童原文的关键差异

1. **不避讳"丑"**：苏童写"溅满了鹭鸟的灰白稀松的粪便"——AI绝不会主动写粪便
2. **少年心智**：苏童写"我讨厌死者，即使死者是我的父亲"——孩子的直接，没有伦理包袱
3. **意象大胆**：苏童写"太阳犹如破碎的蛋黄"——这个明喻之所以不过度，是因为它只有一个，且服务于整个场景的"破碎/死亡"基调
4. **走神即是性格**：端白在仪式中走神看炼丹炉、看玉如意、数钉子——这不是细节堆砌，而是人物性格（一个不想当王的孩子）的呈现

## 段落 2：射杀杨松

### AI版本关键问题

| 问题类型 | AI文本 | 问题说明 |
|---------|--------|---------|
| 替读者解释 | "零下二十度，人的内脏还是热的" | 最典型的"话说死"——把读者自己应该感受到的东西用数据和结论讲出来 |
| "不是/而是"变体 | "不是爬，更像某种虫子在地面上拖着自己往前蹭" | 又是先否定再肯定的纠错模式 |
| 明喻泛滥 | "车厢像一口颠簸的棺材""像烧剩下的骨头""像捧着刚从河边打上来的鱼" | 三句话三个明喻，三个不同语义域（丧葬/烧焦/捕鱼） |
| 添加无据细节 | "去年秋天他在校场射落了悬铃木上的铜钱，我赏了他一壶酒" | 在给人物"加背景故事"，这是剧本思维不是小说思维 |
| 文艺腔收尾 | "把凤凰关、火光、莜麦地和第三十八具尸体都丢在了身后" | 排比式收尾太工整，像在写诗 |

### 苏童原文的关键差异

1. **数字的力量**："一共是三十七具，数到第三十八具的时候我惊叫起来"——不解释为什么数，不解释数的过程中在想什么，就是数。这个数字本身就是全部的呈现。
2. **"杀"作为一个音节的力量**：苏童写"对着锦衣卫喊出一个短促的不可理喻的音节"——先不告诉你喊的是什么，隔了一段才揭晓是"杀"。这就叫"叙述者不与读者同步"。
3. **燕郎的反应才是真正的收尾**：苏童写燕郎说"陛下好箭法"，然后"燕郎的小脸充满了惊惧和谄媚掺杂的表情"——这个表情把一切都收住了。不需要"把一切都丢在身后"这种总结。
4. **不写箭的轨迹，只写结果**："其中一箭异常精确地插入杨松的胸前"——箭怎么飞的不重要，结果才重要。

## 段落 3：蕙妃被诬产狐

### AI版本关键问题

| 问题类型 | AI文本 | 问题说明 |
|---------|--------|---------|
| 明喻泛滥 | "像雨前的燕子""像陈年的米""像溺水的人摸到了船帮""细得像算筹" | 四个明喻又来自四个不同语义域（动物/粮食/溺水/数学），彼此毫无关联 |
| "不是/而是"变体 | "不是攀，是一把攥住" | 再次出现纠错式叙述 |
| 添加无据细节 | "从鹂鸣阁的梁上落下那只死雀开始""最上面一张写的是'夜半来'" | 给人物添加具体诗句/前史，制造不属于原文的"文艺感" |
| 细节过密 | "腕骨突出得像要刺穿皮肤""一节节脊骨，细得像算筹""一只泥金妆盒。盒角磨掉了漆，露出底下的木胎" | 堆砌身体细节+物品细节，但没有一个作为核心贯穿 |
| 文艺腔收尾 | "露水从琉璃瓦上滑下来，打在我的靴面上，隔了很久，又一滴" | 收尾太长太重，像是故意设计出来的诗意 |

### 苏童原文的关键差异

1. **"纸人"意象的贯穿**：苏童从前面就建立了"纸人"这个意象（"蕙妃就像一个美丽的纸人，一半在我怀中，一半却在飘荡"），在此处再度出现（"我觉得她们同样也是两片纸人儿"）。意象的核心不是一次性比喻，而是反复出现、逐渐加深。
2. **对话中的人物性格**：蕙妃的哭诉"她们竟然说我产下的是一只白狐，一只白狐"——重复"一只白狐"，这就是真实人物在情绪中的语言，不是AI能自然写出的。
3. **彭王后吃樱桃的细节**：在整个紧张场景中，插入彭王后"用竹签挑起果盘里的一颗樱桃，从容优雅地往嘴里送"——一个动作胜过十句对话。这就是呈现的力量。
4. **叙述者承认无力**："我是至高无上的燮王，但我还能有什么办法?"——这句话不文艺，不漂亮，但它是真的。AI更倾向于写漂亮的句子，而非真实的句子。

## 跨段落共性问题总结

### 问题 1：明喻饥渴症
**现象**：AI在每200字左右就会插入一个"像..."的明喻，密度大约是苏童原文的3-5倍。而且喻体来自完全不同的语义域。

**根因**：AI在写作时没有"让事物以自身存在"的意识。每次描写都忍不住跳到另一个领域去找喻体。

### 问题 2：叙述者的成人化
**现象**：AI写的叙述者不像一个14岁的孩子。AI的叙述者会：回顾性理解（"后来我才明白"）、做出文艺化的比喻、进行知识性总结。

**根因**：AI默认以"有洞察力的成年叙述者"为写作模式，无法模拟一个14岁孩子有限的理解力和感受方式。

### 问题 3：结尾综合征
**现象**：AI倾向于给每个段落一个"漂亮的、诗意的、有韵味的"结尾。像一个精心设计的退场。

**根因**：AI将段落视为"作品"而非"叙述流的一段"。苏童的段落结尾往往是干脆的动作或对话，不是景观描写。

### 问题 4："纠正"思维残留
**现象**：尽管提示词明确禁止"不是...而是"，但AI以"不，不是X / 不，更像Y"的形式绕过了禁止。

**根因**：深层问题不是句式本身，而是AI在写作中持续进行"自我审视和自我修正"，这是分析性思维的根本特征。需要从思维层面而非句式层面解决。

### 问题 5：干净化倾向
**现象**：AI回避写粪便、臭味、呕吐物、蛆虫等"丑"的细节，即使场景需要它们。

**根因**：AI的训练数据中，"好的中文"更倾向于雅化表达。但苏童的力量恰恰在于不雅化、不回避。
