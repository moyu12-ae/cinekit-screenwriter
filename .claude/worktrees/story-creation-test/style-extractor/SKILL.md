# 文章风格提取器

从多篇中文文学文本中提取风格特征，通过跨文本对比提炼共性，产出结构化的**共性风格画像**。画像是 V9 小说写作流水线中导演 Agent 质感分析环节的标准化输入。

## 核心原则

1. **例文驱动**：每项风格判断必须以原文段落引用为证据，不做无引用的判断
2. **共性优先**：目标是提炼多篇例文中反复出现的共同特征，而非单篇的特殊表现
3. **最小输入**：至少需要 2 篇例文（每篇 ≥ 300 字）才能产出有意义的共性画像

## 使用流程

```
阶段零：例文预处理 → 【检查点1：用户确认例文清单】
阶段一：句式节奏分析（含例文引用）
阶段二：用词倾向分析（含例文引用）
阶段三：感官密度与意象类型分析（含例文引用）
阶段四：叙述距离与情感温度分析（含例文引用）
阶段五：共性提炼 → 【检查点2：用户确认共性画像】
```

### 阶段零：例文预处理

在分析开始前，完成以下工作：

1. **确认例文来源**：收集用户提供的所有文本。可以是文件路径（.txt / .md）或直接粘贴的文本
2. **如果文件较大（>5万字）**：从开头、中间、结尾各选取 500-1000 字的代表性段落作为一个例文单元
3. **每篇例文标注**：来源（文件名/作品名/章节）、篇幅字数、是否为代表性段落还是完整文本
4. **评估文本类型**：小说叙事段落 / 散文 / 对话为主 / 动作描写为主 / 景物描写为主。如果例文类型差异过大（如一篇纯景物+一篇纯对话），在分析前向用户说明——某些维度可能因文本类型差异而不具可比性

**边界条件**：
- 例文 < 2 篇：向用户说明"共性提炼需要至少 2 篇例文"，依然可以做四维度分析，但最终画像中标注"仅基于单篇，未做跨文本验证"
- 例文过短（< 300 字）：标注该例文在"意象类型""感官密度"等维度上可能无法充分判断
- 某篇例文明显异质（风格与其他篇差异极大）：不强行归入共性，在共性提炼中单独标注

**检查点1**：完成预处理后，向用户展示例文清单（含来源、篇幅、类型），等待用户确认后再进入分析。

## 分析流程

按以下五个阶段逐项执行。**每阶段必须为每篇例文引用完整原文段落（5-10句）作为证据**。

四个分析维度的产出遵循统一模板：

```
## [维度名称]

### 例文A：[来源标识]
> [完整段落引用，5-10句]
[该维度各分析要点的逐一分析]

### 例文B：[来源标识]
> [完整段落引用，5-10句]
[该维度各分析要点的逐一分析]

### [维度]共性
[从A和B中提炼的共同模式，逐项列出]
- 差异标记：[值得注意的不同之处]
```

### 阶段一：句式节奏

分析语言的时间性特征。各篇分别引用段落后，按以下要点分析：
- **句长分布**：长句为主还是短句为主？长句的复合句嵌套层数？短句最短几个字？比例？
- **停顿习惯**：逗号密集/稀疏？句号间隔多远？分号/破折号/冒号的使用制造了什么节奏感？
- **句子运动**：线性推进还是回旋推进？是否有"中长句建画面→短句收束"等节奏模式？
- **呼吸感**：绵长低沉 / 短促跳动 / 快慢交替 / 持续的慢？读者在何处被允许停顿、何处被推动？

### 阶段二：用词倾向

分析词汇选择的系统偏好。各篇分别引用段落后，按以下要点分析：
- **口语 vs 书面**：偏向日常口语还是书面雅言？词汇来源领域（庙堂/市井/自然/身体/学术）？
- **词性偏好**：名词多还是动词多？形容词精准克制还是密集堆叠？副词频率和类型？
- **色彩词**：频率和色调偏好（暖/冷/灰/暗/明亮）
- **触觉词与听觉词**：直接描写还是通过意象暗示？
- **修饰语策略**：一个精准的还是三四个叠加的？通用还是出人意料？

> 进阶角度：词的选择不仅是语义选择，还涉及"背景与包晕"（词携带的历史/方言/语体色彩）和"造形的表现"（词在视觉、触觉、运动感觉三个感官通道上的造型能力）。

### 阶段三：感官密度与意象类型

分析感官调动和意象组织。各篇分别引用段落后，按以下要点分析：
- **感官配比**：视觉/听觉/触觉/嗅觉/味觉各占百分比（凭整体感受估算）
- **感官密度**：每百字感官描写次数，密集交织还是稀疏分布？
- **意象的结构性**：各自独立（列举式）还是内在关联（对位式——在意象对立轴上运作）？

### 阶段四：叙述距离与情感温度

分析叙述者与叙述世界的空间和情感关系。各篇分别引用段落后，按以下要点分析：
- **叙述距离**：贴着皮肤的显微镜 / 中距离 / 远距离的望远镜？距离何时拉近/拉远？
- **情感温度**：冷/温/热？叙述者流露情感判断还是压在物象之下？
- **"看到"vs"感到"**：记录视觉事实还是表达内心感受？"我看到"中有没有隐含情感？
- **叙述者介入度**：不介入（只记录）/ 适度介入 / 高度介入（频繁点评）？
- **与读者的关系**：展示（让读者自己看）还是告知（把结论递给读者）？留白多还是写满？

> 进阶角度：叙述距离本质上与"焦距"概念对应——内聚焦（贴着心理看）=近距离，外聚焦（只看行为）=远距离，零聚焦（全知）=可变距离。

### 阶段五：共性提炼

在所有四维度分析完成后执行。不是简单的逐项取交集，而是识别哪些特征是**跨例文稳定出现的结构性模式**，哪些只是单篇的偶然表现。

**共性判定标准**：
- 特征在 ≥ 半数例文中明确出现 → 归入共性
- 特征仅在个别例文中出现 → 标记为"单篇特例"，不在共性画像中呈现
- 特征在所有例文中均未出现但在某篇中缺失 → 检查是否因文本类型差异导致，如是则在"体裁限定"中标注

**共性提炼步骤**：
1. 纵向归并：将四个维度的"共性"区块横向排列，找出跨维度的关联
2. 提取核心张力：从共性中识别最稳定的一组对立或矛盾
3. 写作建议：基于共性特征给出 2-3 条可操作建议

---

## 最终产出：共性风格画像

在所有五个阶段完成后，输出总览块。

### 输出格式

```
## 风格画像：[作者/作品群]

> 基于 [N] 篇例文的共性提炼：[例文来源列表]

### 句式节奏
[共性的总结]

### 用词倾向
[共性的总结]

### 感官密度与意象类型
[共性的总结]

### 叙述距离与情感温度
[共性的总结]

### 核心张力
[从共性中识别的最稳定的对立或矛盾——例如"近距离的视觉记录与冰冷的情感温度之间的张力"、"美和丑在同一句内并置的冲动"、"长句的沉浸与短句的突然截断"等。这是导演 Agent 在规划每个场景的质感时最重要的参考。]

### 单篇特例
[仅在个别例文中出现、未归入共性但值得注意的特征。如无则写"无"。]

### 写作建议
[基于共性特征，为模仿该风格的写作者提供 2-3 条最关键的策略建议]
```

完整示例见 `references/example_sutong.md` —— 苏童《我的帝王生涯》基于 2 篇例文的共性风格画像。

**检查点2**：输出完整风格画像后，等待用户确认。用户可能要求调整共性判断的严格程度、补充某维度的分析、或增加新的例文重新提炼。

## 用途

- 本 skill 输出 → `V9 三Agent流水线` 导演 Agent 的质感分析输入
- 可与其他文学分析 skill 配合使用：风格画像提供宏观文风判断，修辞分析和叙事学分析提供更细粒度的技法审视

---

## 边界条件速查

| 场景 | 处理方式 |
|------|---------|
| 例文 < 2 篇 | 提示补充，可分析但标注"未做跨文本验证" |
| 例文过短（< 300 字） | 标注该例文在部分维度上可信度降级 |
| 例文类型差异过大 | 分析前提示用户，继续分析但标注体裁差异 |
| 某篇明显异质（离群值） | 不强行归入共性，在"单篇特例"中标注 |
| 缺少足够意象的文本（如纯对话） | 阶段三标注"因文本类型限制，意象判断受限" |
| 文件较大（>5万字） | 取开头/中间/结尾各 500-1000 字代表性段落 |
