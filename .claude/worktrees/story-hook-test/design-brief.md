# Story Design Brief — 咖啡与情绪

## I. Story Design

### 1.1 Controlling Idea
孤立转变为连接，因为主人公选择将感知他人痛苦的能力从自我保护的工具转变为共情的桥梁。

### 1.2 Climax Type
向上结局。咖啡师用能力为一位即将永别的沉默老人调制了最后一杯咖啡，尝到的不是悲伤，而是双向的温暖与宁静。

### 1.3 Inciting Incident
某个周二早晨，咖啡师在品尝自己调制的拿铁时，突然尝到了不属于自己的苦涩——一位刚被裁员的客人留下的情绪残味。

### 1.4 Scene Sequence
- S1: 能力降临 — 喝到陌生人绝望的恐慌
- S2: 确认与逃避 — 验证能力真实存在
- S3: 被迫面对 — 回避策略被打破
- S4: 能力升级 — 尝到孩子的纯粹快乐，发现能力不只传苦
- S5: 试炼 — 主动为沉默老人调制咖啡
- S6: 高潮 — 老人的最后一杯，双向宁静
- S7: 新平衡 — 能力仍在，不再是负担

### 1.5 Antagonist Forces
正向：内心平静 → 相反：被迫承受他人情绪 → 矛盾：情绪边界溶解 → 否定之否定：利用能力操纵他人（主人公未滑入的深渊）

## II. Character Design

### 2.1 Protagonist
咖啡师：表面温和有礼，实际用礼貌推开一切连接。定义维度：渴望被理解却恐惧连接。调制咖啡是唯一的表达出口。

### 2.2 Foils
- 被裁员客人（S1）：激发者
- 年轻母亲与孩子（S4）：快乐对照
- 沉默老人（S5-S6）：准导师，每天同一位置点黑咖啡的常客

### 2.3 Character Arc
成长弧：封闭、控制 → 接纳脆弱、以能力为他人创造连接

## III. Dialogue Design
- 咖啡师：借咖啡味道描述情绪
- 老人：极简，每次不超五个字，长停顿
- 高潮节拍：调制(奉献) → 接杯(辨认) → 啜饮(接纳) → 互尝(被回馈) → "够了"(原谅自己) → 目送(告别旧自我)
