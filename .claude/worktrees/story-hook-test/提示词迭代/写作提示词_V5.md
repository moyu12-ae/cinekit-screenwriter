# 小说写作提示词 V5（终版）

## 写作流程：你必须分两步走

### 第一步：写初稿
用第一人称写一段小说文字。你的叙述者是「我」——一个14岁的少年帝王。写在当下，不是回忆录。

### 第二步：强制修改（这一步必须执行，否则整个任务无效）

请拿出你刚写的初稿，逐句执行以下修改：

**修改A**：搜索「像」字。如果出现了超过1次，保留最有力的那1个，其余的全部改写为不依赖比喻的直接描写。改写方法：把「A像B」改成直接写A的颜色、形状、温度、状态。

**修改B**：搜索「不是」。把所有「不是X（，而是Y）」「不，不是」「不是X。是Y」「不是X的，是Y的」中的「不是」及后续否定内容删掉，直接保留肯定内容。例如「不是血滴。是一道一道淌出来的」→ 改为「是一道一道淌出来的」。例如「不是我想数」→ 删掉整句，直接从「数着」开始。

**修改C**：检查结尾。如果最后两句话读起来像「设计过的诗意退场」，把结尾改成一个动作、一句话或一个干脆的短句。

**修改D**：检查对话。确保不同人说的话节奏不同。14岁少年帝王：句子短。哭诉的妃子：一句话反复说，说一半哭一半。老妇人/大臣：句式完整，用词正式。

然后输出修改后的最终版本。

## 场景写作规则（贯穿始终）

1. **叙述者是一个14岁的孩子**：他感受世界的方式是感性的、身体的——他用眼睛看颜色和形状，用皮肤感受温度和质地，用耳朵听声音。他会在重要的场合走神去看炼丹炉的火、去注意别人衣带上挂的玉如意。他的情感表达可以直接到冷酷（「我讨厌死者，即使死者是我的父亲」）。他不分析、不总结、不升华。**重要：不要把所有东西都量化。** 不要把每件事都变成数字——台阶就是台阶，不一定是「十九级台阶」；尸体就是尸体，不一定要数到「第三十七具」。数字偶尔用，用在真正需要强调的时刻，才有力量。大多数时候，感受优先于计数。

2. **不回避丑陋**：鸟粪、血污、肠子的颜色、腐烂的臭味——场景里有什么就写什么。不绕开，不加滤镜。

3. **动作和物象说话**：不写情绪词（愤怒、悲伤、恐惧）。写字条具体的东西：手在做什么、眼睛里是什么、碟子里有几颗樱桃核。

4. **长短句错开呼吸**：不要连续五句相同长度。不要全部碎片短句。长句建画面，短句锤击。

5. **不添背景不加前史**：场景概要里没有的东西不要加。人物不需要被补充履历。
