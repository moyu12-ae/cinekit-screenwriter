# 小说写作提示词 V6 —— 三层决策框架

## 写作流程：策略先行，三步产出

### 第〇步：叙事定位（动笔前必须想清楚）

以下每个问题，先在脑中回答，再开始写。

**A. 叙述者是谁？**
- 在场还是不在场？（叙述者是故事中的人物「我」，还是站在故事之上的全知者？）
- 如果是第一人称「我」：他的年龄、身份、心智水平是什么？他的词汇量有多大的范围？他有什么是他不知道的、看不到的、理解不了的？
- 他的讲述是否有变形？（可靠叙述者还是不可靠叙述者——他是否在美化自己？是否在隐瞒？是否因为认知有限而误解？）

**B. 视点、焦点、焦距**
- 视点：内视点（站在故事内某个人的位置看）还是外视点（站在外面看所有人的外在行为）？
- 焦点：内聚焦（看内心活动）还是外聚焦（只看言行表面）？
- 焦距：当前场景是显微镜（贴近人物感官，每一点细微变化都可见）还是望远镜（远观，只写大动作）？

**C. 这一段的修辞境界是什么？**
- 陈望道分三个境界：记述（平实叙述，以明白为目标）、表现（以感染为目标，积极修辞大量使用）、糅合（两种混合）。
- 小说中的过渡/交代段落用记述境界（消极修辞为主，平实清晰）。
- 小说中的核心场景用表现境界（积极修辞为主，唤起读者感受）。
- 不要在记述段落滥用积极修辞（如用比喻去交代「他很累」），也不要在该表现的地方只做干枯叙述。

### 第一步：写初稿

带着第〇步的定位，用第一人称写一段小说文字。

### 第二步：强制修改

拿出初稿，逐句执行以下修改：

**修改A——譬喻控制**：
- 搜索「像」「仿佛」「如同」「犹」「若」「如」「似」。如果明喻超过1个，保留最有力的1个，其余全部改为直接描写。
- 检查唯一保留的那个明喻：正文和譬喻是否在一点上极其类似，在整体上极其不同？（陈望道所谓譬喻成立的两个必要条件）
- 检查譬喻的素材是否来自叙述者能体验到的日常世界？（修辞素材的体验基础：观察性、感受性、经验性、生活性）

**修改B——消除纠正句式**：
- 搜索「不是」。把所有「不是X（，而是Y）」类句式中的否定部分删掉，直接保留肯定内容。

**修改C——检查结尾**：
- 如果最后两句话读起来像设计过的诗意退场，改成一个动作、一个干脆的句号。

**修改D——检查摹状**：
- 是否有直写声音的摹声辞（如「嗒嗒」「沙沙」）？它们是否融入了句子而非单独加叹号？
- 是否有「借感觉表现气氛」的摹状？如果有，它们是否有力量让人同感？没有就删。

**修改E——检查对话**：
- 不同人说话节奏不同。叙述者的第一人称声音保持一贯。

## 场景写作规则（贯穿始终）

1. **叙述者的感知方式**：他用眼睛看颜色和形状，用皮肤感受温度和质地，用耳朵听声音。他在重要场合会走神。他的情感表达可以直接到冷酷。他不分析、不总结、不升华。感受优先于量化计数。

2. **消极修辞为底子**：即使是表现段落，也要保证意义明确、语句通顺、词句平匀、安排稳密。不要为了「文学感」制造不明不白的句子。

3. **积极修辞为感染**：譬喻（明喻/隐喻/借喻）、摹状（摹声/摹感觉）是工具而非目的。用它们是为了让读者感受到，不是为了展示文采。

4. **素材的体验基础**：所有修辞素材必须来自叙述者能观察、能感受、能经验的日常世界。不用科学术语做喻体，不用书本知识代替身体感受。

5. **长短句错开呼吸**：长句建画面，短句锤击。不要连续五句相同长度。
