# 小说写作提示词 V7 —— 三层框架 + 强制修訂

## 写作流程（必须严格按顺序）

### 第〇步：叙事定位（动笔前先想清楚）

阅读下面的策略卡。它告诉了你叙述者是谁、用什么视点、焦距多远、场景的修辞境界是什么、节奏怎么设计。

### 第一步：写初稿

用第一人称写一段小说文字。

### 第二步：强制修改（这一步不能被跳过）

拿出初稿，按以下顺序逐项执行：

**第一项：譬喻清查（最重要）**

1. 数出初稿中所有的明喻（带「像」「仿佛」「如同」「犹」「若」「如」「似」的句子）：__个。
2. 如果大于1个，只保留最有力的那1个。其余全部改为：
   - **直接摹写**：写事物本身的颜色、形状、温度、状态（陈望道所谓「摹状」）
   - 或者升为**隐喻**（去掉「像」，直接说「甲就是乙」。例如「她的骨头是算盘珠子」——更紧切，更有力）
   - 或者升为**借喻**（正文也不出现，直接用譬喻物做代表。例如不说「她瘦得像纸人」，而是直接写「纸人在风中飘荡」）
   
   陈望道的譬喻三级规律是：**越进一级，形式越简短，譬喻的客体越升到主位**。当感情强烈时，用更高级的譬喻（隐喻/借喻）比明喻更有效。

3. 检查唯一保留的譬喻：
   - 正文和譬喻在一个点上极其类似吗？
   - 在整体上极其不同吗？
   - 譬喻素材来自叙述者能观察、能感受、能经验的日常世界吗？

**第二项：纠正句式清查**

搜索「不是」。把所有「不是X（，而是Y）」「不，不是X」「不是X。是Y」「准确地说」「与其说X不如说Y」中的否定部分删掉，直接保留肯定内容。

**第三项：结尾检查**

如果最后两句读起来像设计过的诗意退场，改成一个动作、一句话、一个干脆的句号。

**第四项：对话检查**

不同人说话的节奏不同。少年帝王：句子短，直接。哭诉者：一句话反复说，说半句哭半句。

然后，输出修改后的最终版本。

## 场景写作规则

1. **叙述者在感受**：眼睛看颜色形状，皮肤感受温度质地，耳朵听声音。会走神。不分析、不总结、不升华。感受优先于量化。

2. **消极修辞做底子**：保证意义明确、语句通顺、词句平匀、安排稳密。

3. **积极修辞为感受服务**：譬喻和摹状是工具不是目的。用它们是为了让读者感受到，不是为了展示文采。当感情强烈时，用隐喻/借喻替代明喻。

4. **素材的体验基础**：所有修辞素材来自叙述者能观察、能感受、能经验的日常世界。

5. **长短句错开呼吸**。
