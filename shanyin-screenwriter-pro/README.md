山音超级编剧大师 - 全格式影视编剧 Skill
由 @山音 设计的自然语言驱动编剧 Agent Skill
覆盖 1-3 分钟概念超短片→90 分钟电影长片→多集剧集的全流程剧本创作，一站式搞定人物、大纲、场景、剧本全链路

核心能力
全格式覆盖（从短片到长片，无缝适配）
概念超短片：支持 how-to-tell、what-if两大方向，1-3 分钟快速出片
叙事短片：5-10 分钟精品短片，完整起承转合，适配短视频 / 短片竞赛
90 分钟长片：商业片（强剧情、高爽点）& 文艺片（深情感、强质感）双轨创作
多集剧集：支持季播 / 单元剧架构，完成全剧大纲 + 单集剧本 + 角色成长线设计
全流程编剧服务
人物设计：立体角色设定（性格、动机、成长弧光、反差特质）
结构大纲：经典叙事模型（三幕式 / 英雄之旅）+ 类型化框架（悬疑 / 科幻 / 爱情 / 犯罪）
场景拆解：分镜级场景规划（镜头语言暗示、情绪节点、转场设计）
完整剧本：标准影视格式（场景编号、动作描述、对话、转场）精准输出

使用方法
1. 安装 Skill
支持 Agent Skills 标准的 AI 工具（Claude Code、Reasonix Code、Codex 等）：
将整个 `shanyin-screenwriting-master/` 目录放入对应 skills 目录：
- Claude Code：`~/.claude/skills/`
- Reasonix Code：`~/.reasonix/skills/`
- Codex：`~/.agents/skills/`

或放在项目的 `.reasonix/skills/` / `.claude/skills/` 目录下。

2. 快速触发
直接输入自然语言即可触发，任何创作内容均可交互。
3. 进阶定制
可主动向 Agent 补充细节（提升剧本精准度）：
明确受众（如青少年 / 成年观众 / 海外市场）
指定风格标签（如韦斯・安德森质感、诺兰悬疑风、宫崎骏治愈风）
要求适配特定平台（如短视频平台节奏、院线电影时长、剧集周播逻辑）
