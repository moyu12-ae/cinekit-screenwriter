## 诊断清单

### 自检一：写作红线
- [ ] 无心理描写（"他意识到""她领会到"）
- [ ] 无括号暗示（"（其实是在掩饰紧张）"）
- [ ] 无角色用台词解释设定
- [ ] 无说教片段
- [ ] 无反例速查中的 5 个高频错误

### 自检二：结构完整性
- [ ] 每场戏有价值翻转
- [ ] 因果链不断裂
- [ ] 障碍层层递进
- [ ] 不归点有力
- [ ] 开场钩子有效

### 自检三：人物一致性
- [ ] 每个角色有独特的"句型模板"
- [ ] Want/Need 张力贯穿全片
- [ ] 弧光有伪变化→真正转变的递进
- [ ] 配角功能明确

### 自检四：对白质量
- [ ] 遮住名字能分辨谁在说话
- [ ] 每句对白有潜文本（动名词可标注）
- [ ] 无"如你所知"式科普对白
- [ ] 对白锚点有三级进化

### 自检五：节奏
- [ ] 无连续 3 场以上同节奏
- [ ] 高潮前有蓄积
- [ ] 有喜剧 relief / 情感呼吸点

### 自检六：信息设计
- [ ] 设定信息通过冲突揭示
- [ ] 伏笔有回扣计划
- [ ] 无信息dump段落
